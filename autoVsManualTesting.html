Auto Testing

What is automated testing? Automated-testing is when you are using an automated tool to execute a test on the software you have. The advantage of automated software testing is that the machine can also enter test data it has received. Some automated tests can compare expected and actual results and generate detailed test reports. Test Automation demands considerable investments of money and resources.
